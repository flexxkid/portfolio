The Person Behind the Code

Built on
real curiosity.

Not every developer starts with a clear destination. This is where I came from, what drives me now, and where I'm headed.

Origin

I didn't start in Computer Science; I began my university journey in a completely different field. But everything changed in 2021 when I started experimenting with mobile compilers on my phone. Writing code on a tiny screen and watching logic execute in real-time sparked something immediate. That raw curiosity grew into an obsession, driving me to pivot entirely and pursue a formal path. I found exactly what I was looking for at Kenyatta University: a rigorous foundation to back up my self-driven start (Sept 2022).

Now

To me software and networking are two sides of the same coin. Right now, my focus is on this exact convergence. I engineer full-stack web applications with a network engineer’s eye, using Cisco Packet Tracer to simulate the exact corporate environments my code will live in, and Wireshark to dissect packet traffic at a granular level. It’s about building software that doesn't just run perfectly on a local screen, but thrives securely across a complex network.

Direction

Building competence in both development and networking is an intentional step toward my long-term goal in Cybersecurity. Effective digital defense requires a clear understanding of how applications are constructed and how data travels between them. By mastering these two foundational disciplines first, I am equipping myself with the practical skills needed to eventually transition into cybersecurity architecture; ensuring I can build systems that are resilient and secure from the ground up.

How I Think About Software
  • 01 Reliability over novelty. The most elegant code is worthless if it drops connections or crashes under load. I build for uptime and predictable performance.
  • 02 Maintainability is financial efficiency. Well-documented, clean architecture means future developers can scale the system quickly without wasting billable hours rewriting it.
  • 03 Security is non-negotiable. Protecting user privacy and operational data isn't a premium feature; it is an baseline engineering requirement for every project I touch.
